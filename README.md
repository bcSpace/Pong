# Pong
---

# Goal 
The goal of this project was to create a 2 player game of pong 

# What it does
Can either play 1v1 with 2 local players or watch 2 A.I players play against each other

# How to play
Main Menu: Arrow keys and enter

Playing locally: W/S up and down for player 1, UP/DOWN for player 2

Watching AI play: No controls

Space: Pause
